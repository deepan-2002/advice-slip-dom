## Usage

### Development

```bash
npm run dev
```

Just run and visit http://localhost:5173


### Build

To build the App, run

```bash
npm run build
```

And you will see the generated file in `dist` that ready to be served.

### Preview

To preview the build files, run

```bash
npm run preview
```
